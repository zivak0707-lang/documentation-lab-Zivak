# To-Do App — Трекер завдань

**Коротко:** To-Do App — простий веб-додаток для ведення списку завдань (список справ). Допомагає користувачу планувати день, додавати, редагувати, відмічати виконані пункти та фільтрувати їх.

**Яку проблему вирішує:** дозволяє зручно контролювати завдання, не забувати важливе та розбивати роботу на кроки.

## Основні функції
- Додавання нового завдання (title, опис, пріоритет).
- Редагування та видалення завдань.
- Позначення як виконане / невиконане.
- Фільтрація: всі / активні / виконані.
- Пошук по заголовку.

## Для кого продукт
- Студенти, які хочуть організувати навчання.
- Люди, що планують щоденні справи.
- Новачки, які хочуть практикуватися з простим веб-проєктом.

## Структура репозиторію
- `README.md` — загальний опис (цей файл).
- `USER_GUIDE.md` — інструкція для користувача.
- `SETUP_GUIDE.md` — як встановити та запустити.
- `API_OVERVIEW.md` — приклад API.
- `CHANGELOG.md` — журнал змін.
- `images/` — ілюстрації інтерфейсу.
