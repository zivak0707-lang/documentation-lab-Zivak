# To-Do App — Трекер завдань (documentation-lab-Zivak)

**Автор:** Zivak (GitHub: @zivak0707-lang)
**Коротко:** To-Do App — простий веб-додаток для ведення списку завдань. Допомагає планувати день, додавати, редагувати, відзначати виконані пункти та фільтрувати їх.

**Яку проблему вирішує:** дозволяє зручно контролювати завдання, не забувати важливе і легко планувати робочі/навчальні дії.

## Основні функції

* Додавання нового завдання (title, опис, пріоритет).
* Редагування та видалення завдань.
* Позначення як виконане / невиконане.
* Фільтрація: всі / активні / виконані.
* Пошук по заголовку.

## Для кого продукт

* Студенти, які організовують навчання.
* Люди, що планують щоденні справи.
* Новачки, які хочуть практикувати розробку простих веб-проєктів.

## Структура репозиторію

* `README.md` — загальний опис (цей файл).
* `USER_GUIDE.md` — інструкція для користувача.
* `SETUP_GUIDE.md` — інструкція зі встановлення та запуску.
* `API_OVERVIEW.md` — приклад технічної документації (endpoints).
* `CHANGELOG.md` — журнал змін.
* `images/` — ілюстрації інтерфейсу (завантажити зображення сюди).
