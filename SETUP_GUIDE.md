# Посібник зі встановлення — To-Do App

> Інструкція для запуску демо-версії локально. Наведені кроки — адаптовані для початківця.

## Необхідні інструменти

* Веб-браузер (Chrome, Firefox).
* Node.js (рекомендовано версія 14 або вище) — тільки якщо проект має фронтенд на Node.
* Git — опціонально (для клонування репозиторію).
* Текстовий редактор (VS Code або Notepad).

## Кроки: як отримати код і запустити (детально)

### A. Якщо ти працюєш через GitHub веб (рекомендовано для початківців)

1. Відкрий репозиторій `https://github.com/zivak0707-lang/documentation-lab-Zivak`.
2. Натисни кнопку **Code** → вибери **Download ZIP** → натисни **Download**.
3. Відкрий завантажену папку (розпакуй ZIP).
4. Якщо в папці є інструкція `package.json` (фронтенд):

   * Відкрий термінал у цій папці і введи `npm install`, потім `npm start`.
   * В браузері відкрий `http://localhost:3000`.

### B. Якщо ти знаєш Git і хочеш клонувати

1. Скопіюй URL → натисни **Code → HTTPS → Copy**.
2. У терміналі:

```bash
git clone https://github.com/zivak0707-lang/documentation-lab-Zivak.git
cd documentation-lab-Zivak
# якщо є фронтенд:
npm install
npm start
```

## Перевірка роботи (що перевірити)

* Відкрий `http://localhost:3000` → бачиш поле для введення нового завдання? → Додай тестове завдання → воно з’явилось у списку — значить все працює.
* В терміналі при запуску має з’явитися повідомлення: `Compiled successfully` або `Server running at http://localhost:3000`.

## Розв'язання типових проблем

* **npm: command not found** — встанови Node.js з [https://nodejs.org](https://nodejs.org).
* **Port 3000 is already in use** — закрий інші процеси або зміни порт (параметри скрипту).
* **404 при зверненні до API** — перевір, що бекенд запущено і URL вірний.
